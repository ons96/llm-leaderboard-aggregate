# Leaderboards outputs (Phase 2)

This directory is generated by the benchmark aggregation pipeline (`python -m modeldb_builder.benchmarks`).

## Files

- `agentic_coding_leaderboard.csv`: One row per canonical model (from `models_unique`), ranked by `avg_agentic_coding_score` descending.
- `reasoning_chat_leaderboard.csv`: One row per canonical model, ranked by `avg_reasoning_chat_score` descending.
- `gateway_fallback_ranking.csv`: One row per `(model_id × provider_name × provider_model_id)` (from `model_providers`), ranked by `provider_score` descending.
- `leaderboard_metadata.json`: Run metadata (source status, row counts, coverage counts).

## Score definitions

### `avg_agentic_coding_score` (model-level)

Normalized weighted geometric mean over available sources (missing sources are ignored):

- SWE-bench Verified `%` (40%)
- LiveCodeBench `%` (25%)
- SWE-rebench `%` (20%)
- LM Arena `arena_elo_coding` normalized to 0–100 (15%)

### `avg_reasoning_chat_score` (model-level)

Normalized weighted geometric mean over available sources (missing sources are ignored):

- LiveBench `reasoning` score (35%)
- LiveBench `overall` score (30%)
- LM Arena `arena_elo` normalized to 0–100 (35%)

### `provider_score` (provider-level)

Used for LLM gateway fallback ordering:

provider_score = avg_agentic_coding_score
